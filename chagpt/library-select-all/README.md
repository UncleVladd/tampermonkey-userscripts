# ChatGPT Library Select Current View

A Tampermonkey userscript that adds **Select current view** and **Clear selection** buttons to the ChatGPT Library.

It selects only files currently loaded on the page. It does not delete files automatically.

## Installation

1. Install the Tampermonkey browser extension.
2. Open the Tampermonkey dashboard.
3. Click **Create a new script**.
4. Delete the default script contents.
5. Paste the contents of `chatgpt-library-select-all.user.js`.
6. Save with `Ctrl+S`.
7. Refresh ChatGPT and open the **Library** page.

## Usage

1. Open ChatGPT Library.
2. Click **Select current view**.
3. Review the selected files.
4. Click ChatGPT’s native **Delete** button.

Use **Clear selection** to deselect the files.

Files that have not yet loaded will not be selected. Scroll down to load more files, then click **Select current view** again.
